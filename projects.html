オープンソース・ソフトウェア

  • isogram
    Node
    任意の引数名を持ったGoogle Analyticsスニペットを出力するツール。
  • uglify-save-license
    Node
    UglifyJSを用いてソースコードを圧縮する際にライセンスを含むコメントを判別し、残しておくことができるモジュール。通常の圧縮ではライセンスを残すことが困難なUnderscore.jsやBackbone.jsにも対応しています。
  • grunt-merge-data
    Node
    複数のJSONまたはYAMLファイルのデータを単一のデータに統合するGruntプラグイン。統合したデータはJSONファイルとして出力できるほか、Gruntコンフィグに直接代入することも可能です。
  • flex-svg
    Node
    SVGファイルからwidth属性とheight属性を削除するモジュール。SVGグラフィックのサイズをCSSから決定したい場合に役立ちます。Grunt用およびgulp用のプラグインも作成しました。
  • AudioContext-Polyfill
    JavaScript (client-side)
    Web Audio API用のポリフィル。ベンダープレフィックスやAPI名の変更を吸収し、最新のAPIでWeb Audioのコードを書くことができます。
  • Broccoli プラグイン
    Node
    Node製ビルドツールの一つ、Broccoli用のプラグイン。clean-css、TypeScript、LiveScript、IcedCoffeeScript、esformatter、CSS MQPackerのプラグインを作成しています。
