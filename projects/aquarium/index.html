Twinkle Aquarium
2013|グラフィックプログラミング / 映像撮影・編集
映像
https://vimeo.com/88578221
Github
https://github.com/shinnn/twinkle-aquarium

魚群の動きを再現した映像を鑑賞できるMac OS X用アプリケーションです。マイクからの音声入力を行っており、音が鳴るとそれに反応して逃げ去るような動きをします。

魚群はボイド理論に調整を加えたアルゴリズムに基づいて集合、整列、分散します。

2013年8月下旬から9月上旬にかけて開催された新宿クリエイターズフェスタに出展し、京王プラザホテル1階通路の壁面にプロジェクションするかたちで展示を行いました。プロジェクターを5台使用し、全長約30mのスペースの中で魚群を投影しました。公共空間での展示であり、通行者の興味を早く引き、短時間での鑑賞でも楽しんでもらえるようにするため、「魚の群れが、通行者の発する音に驚いて逃げる」という、シンプルなインタラクションを採用しました。
