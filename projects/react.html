re(a)ct
2013|フロントエンド開発 / UIデザイン / 楽曲・効果音制作
サイト
http://shinnn.github.io/re-a-ct/
Github
https://github.com/shinnn/re-a-ct

ブラウザで動作するリッチオーディオコンテンツを実現するための手段として期待されるHTML5 Web Audio APIの習作として制作したWebアプリケーションです。クリックした位置にキャラクターが生成され、BGMに合わせて動き廻ります。キャラクター同士がぶつかると別の音が発生し、そのたびにキャラクターが成長していきます。

リズムに合わせた操作を実現するために、Gruntでのビルドの際にaubioを用いたビートディテクションを行い、リズム情報をJSONデータとして書き出し利用しています。
